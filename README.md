# Installation
- clone project
- cd project directory
- `nvm use`
- `npm i`

# Local Development
- `npx localtunnel --port 3001`
- copy tunnel URL
- cd project directory
- `export HOST={YOUR_LOCALTUNNEL_URL}/bsky-oauth-playground/ PORT={YOUR_PORT}`
- `npm run dev`
